# Fingerspelling

Fingerspelling is spelling a word letter-by-letter, borrowing from the spoken language alphabet.
This phenomenon, found in most signed languages, often occurs when there is no previously agreed-upon
sign for a concept, like in technical language, colloquial conversations involving names,
conversations involving current events, emphatic forms, and the context of code-switching between the signed language
and the corresponding spoken language.

### Data

The SignWriting fingerspelling is mostly taken from: https://www.signwriting.org/forums/software/fingkeys/fkey001.html
The transcribed was performed by Sutthikhun Phaengphongsai, paid for by `sign.mt ltd`.

Every file in the data directory is named:
`LANGAUGE`-`COUNTRY`-`SIGNED`-`NAME`.txt

Where:

1. `LANGUAGE` is a 2 letter `ISO 639-1` code
2. `COUNTRY` is a 2 letter `ISO 3166` ALPHA 2 code
3. `SIGNED` is a 3 letter `ISO 639-3` code from
   the [language subtag registry](https://www.iana.org/assignments/language-subtag-registry/language-subtag-registry)
4. `NAME` is a non-unique sign language code, as known locally
